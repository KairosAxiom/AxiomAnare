## Two changes, bundled — but VERIFY BEFORE PUSH

This package contains two distinct things:
1. **Assumptions feature** (parser + app.js render/prompt) — tested, ready.
2. **ISO 10816-3 CONFIG correction** (app.js values + citations) — verified against

## Suggested commits (two, for a clean history)

Because the two changes are separable, two commits read better in `git log` — but
they touch the same `app.js`, so if you prefer one commit that's fine; just use
the combined message at the bottom.

```

Generalise A12 no-silent-defaults beyond sample rate. Parser publishes

app.js merges with its own sample-rate provenance into one consolidated
```
